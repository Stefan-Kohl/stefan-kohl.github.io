My contributions to the GAP Library

  • Constructors for the Suzuki groups Sz( q ) as
    • matrix groups and as
    • permutation groups, permutation-isomorphic to the action of Sz( q ) on Tits' ovoid.
    Since version 4.2.
  • Functions and methods for the efficient computation of class numbers of
    • GL( n, q ),
    • the groups isogeneous to SL( n, q ),
    • PSL( n, q ),
    • GU( n, q ),
    • the groups isogeneous to SU( n, q ), and
    • PSU( n, q ).
    Since version 4.3.
  • Constructors for
    • GL( n, Z/mZ ) and
    • SL( n, Z/mZ ).
    Since version 4.3.
  • Functions for computing
    • continued fraction expansions and
    • continued fraction approximations
    of real roots of polynomials with integer coefficients.
    Since version 4.4.5.
  • A method for computing structure descriptions for finite groups.
    Since version 4.4.5.
  • A function for computing partial factorizations of integers, which permits the user to specify the amount of work he wants to invest in finding factors.
    Since version 4.4.6.
